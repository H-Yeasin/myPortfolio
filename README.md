# H. U. Yeasin - Portfolio

A premium, modern portfolio application built with Flutter, showcasing engineering expertise with a focus on Educational Technology (EdTech) and interactive user experiences.

![Banner Placeholder](assets/images/banner.png)

## 🚀 Overview

This portfolio is not just a showcase of projects; it's a demonstration of a philosophy. With a background in **Educational Technology (B.Sc.)**, I build applications that prioritize cognitive efficiency, inclusive design, and robust architecture.

## ✨ Key Features

- **Dynamic Hero Section**: Interactive and engaging introduction to the developer.
- **EdTech Philosophy**: A dedicated section explaining the instructional design principles behind the development process (Cognitive Load Theory, LMS Architecture, Inclusive Design).
- **Project Showcase**: A responsive grid of selected projects with detailed views, categories, and technologies.
- **Technical Writing**: Integration for sharing insights and articles.
- **Playground**: A space for experimental widgets and micro-interactions.
- **Adaptive UI**: Seamless transitions between Light and Dark modes with glassmorphism aesthetics.

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev)
- **State Management**: [Riverpod](https://riverpod.dev)
- **Animations**: [Flutter Animate](https://pub.dev/packages/flutter_animate)
- **Typography**: [Google Fonts](https://pub.dev/packages/google_fonts)
- **Architecture**: Clean Architecture principles within feature-first structure.

## 🛠️ Getting Started

### Prerequisites

- Flutter SDK (latest stable)
- Dart SDK
- Android Studio / VS Code with Flutter extensions

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Yeasin84/myPortfolio.git
    cd myPortfolio
    ```

2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run the application:**
    ```bash
    flutter run -d chrome # For web development
    # OR
    flutter run # For mobile/desktop
    ```
